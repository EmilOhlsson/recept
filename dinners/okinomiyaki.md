# Okinomiyaki
Vitkålspannkaka
```
400 g vitkål

4 st ägg
4 dl vetemjöl
3 dl vatten
salt, peppar
```
Okinomiyaki-såsen
```
1.5 dl worcestershiresås
0.5 dl japansk soja
2 tsk mald torkad ingefära
2 msk vitvinsvinäger
2 msk socker
1 tsk arrowroot/maizena/potatismjöl
```
Tillbehör
```
Majonnäs
salladslök
```
## Gör så här
* Blanda sås
* strimla vitkål tunnt
* Blanda pannkakssmet
* Stek vitkål, häll på pannkakssmet
